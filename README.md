# Zenith Digital - Unreal Engine Development Website

A professional website built with Next.js and Tailwind CSS for showcasing Unreal Engine development services.

## Features

- **Modern Tech Stack**: Built with Next.js 15, TypeScript, and Tailwind CSS
- **Responsive Design**: Mobile-first approach with responsive navigation
- **Professional UI**: Dark theme with blue accent colors suitable for gaming/tech industry
- **SEO Optimized**: Proper meta tags and semantic HTML structure
- **Performance**: Optimized for fast loading and smooth user experience

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

```
src/
├── app/
│   ├── components/
│   │   └── Navigation.tsx     # Main navigation component
│   ├── globals.css            # Global styles with Tailwind
│   ├── layout.tsx             # Root layout with navigation
│   └── page.tsx               # Homepage with hero and services sections
```

## Development

- Edit pages in `src/app/page.tsx`
- Add components in `src/app/components/`
- Styles are handled through Tailwind CSS classes
- The page auto-updates as you edit files

## Technologies Used

- **Next.js 15**: React framework for production
- **TypeScript**: Type-safe JavaScript
- **Tailwind CSS**: Utility-first CSS framework
- **ESLint**: Code linting and formatting
